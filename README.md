# AI Image Detector

This project detects whether an image is AI-generated or real. It is being developed for CNIT/CIT 58100 at Purdue University.

## Project Goals
- Build a binary classifier for AI vs real images
- Generate structured JSON explanations
- Prepare the model for mobile deployment with Qualcomm AI Hub

## Files
- `train_baseline.py` - trains the baseline model
- `infer_and_explain.py` - predicts and creates JSON output
- `requirements.txt` - dependencies
- `explanation_schema_example.json` - example output format

## Notes
Datasets and large model files are not stored in this repository.
